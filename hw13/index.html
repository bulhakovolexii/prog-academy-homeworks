Task 1

Побудувати прямокутний трикутник, що складається із символу "*".

Результат → 


    

    
Код:

    
function task1() {
    document.getElementById('result1').innerHTML = '' //сlears the previous result
    let edge = +prompt('Введіть довжину ребра');
    let str = '*'
    for (let i = 0; i < edge; i++) {
        document.getElementById('result1').append(str + '\n');
        str += '*';
    }
}

function task2() {
    document.getElementById('result22').querySelector('span').innerHTML = ''; //сlears the previous result
    const a = +prompt('Введіть число А');
    const b = +prompt('Введіть число В');
    let sum = 0;

Task 3

    
Вивести на екран ряд чисел Фібоначчі, що складається з n елементів (n прийняти від користувача).

    
Числа Фібоначчі - це елементи числової послідовності, в якій кожне наступне число дорівнює сумі двох попередніх.
    

    
    
Результат → 

    
Код:

    
function task3() {
    document.getElementById('result3').querySelector('span').innerHTML = ''; //сlears the previous result
    const n = +prompt('Введіть n');
    let a = 1;
    let b = 1;
    document.getElementById('result3').querySelector('span').append(a + ' ');
    document.getElementById('result3').querySelector('span').append(b + ' ');
    for (let i = 3; i <= n; i++) {
        let c = a + b;
        a = b;
        b = c;
        document.getElementById('result3').querySelector('span').append(b + ' ');
    }
}

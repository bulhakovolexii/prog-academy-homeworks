Task 1

За координатами двох точок, які вводить користувач, визначити рівняння прямої, що проходить через ці точки.

Загальний вигляд рівняння: y = kx + b; де k = (y1 - y2) / (x1 - x2), b = y2 - k*x2.

Результат →

Код:

        function task1() {
            const y1 = +prompt('Введіть координату y1');
            const x1 = +prompt('Введіть координату x1');
            const y2 = +prompt('Введіть координату y2');
            const x2 = +prompt('Введіть координату x2');
            const k = (y1 - y2) / (x1 - x2);
            const b = y2 - k * x2;
            const equation = `y = ${k}x + ${b}`;
            document.getElementById("result1").querySelector('span').innerHTML = equation;
        }

Task 5

Запитайте у користувача його вік, після чого виведіть на екран модальне вікно із запитанням: "Ваш вік ...(введене користувачем число) років?".

Далі, залежно від вибору користувача, виводиться модальне вікно з інформацією true або false.

Код:

        function task5() {
            const age = +prompt('Скільки вам років?');
            const confirmation = confirm(`Вам ${age} років?`);
            alert(confirmation);
        }

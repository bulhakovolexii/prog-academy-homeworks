Task 1

Серед трьох чисел знайти середнє арифметичне. Якщо серед чисел є рівні, вивести повідомлення "Помилка". Числа прийняти від користувача.

Результат →

Код:

        function task1() {
            const a = +prompt('Введіть перше число');
            const b = +prompt('Введіть друге число');
            const c = +prompt('Введіть третє число');
            let result;
            if (a === b || b === c || a === c) {
                alert('Помилка');
            } else {
                result = (a + b + c) / 3;
            }
            document.getElementById("result1").querySelector('span').innerHTML = result;
        }

Task 2

Визначити якесь із трьох, введених користувачем, чисел максимальне і вивести його на екран.

Результат →

Код:

        function task2() {
            const a = +prompt('Введіть перше число');
            const b = +prompt('Введіть друге число');
            const c = +prompt('Введіть третє число');
            let max;
            if (a > b) {
                max = a;
            } else {
                max = b;
            }
            if (c > max) {
                max = c;
            }
            document.getElementById("result2").querySelector('span').innerHTML = max;
        }
